Basic Security

How can you avoid being constantly watched by these sites?

There are different levels of precautions you can take to ensure you aren't being traced constantly by the sites you use.
Some of the basic precautions include:

VPNs

VPNs masks an ip address by creating a private address from a public one. This way, it makes it exponentially more difficult for a user's data to be collected, and overrides any restrictions usually placed on local sites. Some of the most famous VPN services available include: nordVPN, and expressVPN.
Link to tutorial: How to set up a VPN

Basic security measures you can take to leave a smaller footprint.

Use secure browsers like Google Chrome or Microsoft Edge.
Keep antivirus software up to date, and regularly scan your device to ensure no malware is installed erroneously.
Delete site ‘cookies’ periodically, or consider disabling them completely.
